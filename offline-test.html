📡My Chord Sheets - オフライン機能テスト手順

🎯 オフライン対応の仕組み

このアプリはPWA（Progressive Web App）として作られており、以下の技術でオフライン動作を実現しています：

  • Service Worker - バックグラウンドで動作し、ファイルをキャッシュ
  • Cache API - HTML/CSS/JavaScriptを端末に保存
  • LocalStorage - あなたの楽曲データを保存

✅ オフライン機能のテスト方法

ステップ 1: アプリを公開する

GitHub PagesまたはNetlifyでアプリをデプロイ（README参照）

ステップ 2: 初回アクセス（オンライン状態）

デプロイしたURLにアクセス → 全ファイルが自動的にキャッシュされます

開発者ツール（F12）→ Consoleタブで以下のメッセージを確認：
✅ Service Worker登録成功 - オフライン対応完了！

ステップ 3: オフラインモードをテスト

方法A: ブラウザのDevToolsを使用

  1. F12キーで開発者ツールを開く
  2. Networkタブを選択
  3. 「Offline」にチェックを入れる
  4. ページをリロード（F5）

方法B: スマホの機内モード

  1. スマホでアプリにアクセス
  2. 機内モードをONにする
  3. アプリを開く
ステップ 4: 動作確認

オフライン状態でも以下が動作すること：

  • ✅ アプリが開く
  • ✅ 既存の楽曲が表示される
  • ✅ 新しい楽曲を作成・編集できる
  • ✅ キー変更が動作する
  • ✅ ヘッダーに「オフライン」表示が出る
  • ✅ 黄色い「オフラインモード」バナーが表示される
ステップ 5: オンラインに戻る
  1. オフラインモードを解除
  2. ページをリロード
  3. 「オンライン」表示に変わることを確認
  4. オフライン中に作成したデータが残っていることを確認

📱 スマホでのインストールとオフラインテスト

iOS（Safari）
  1. Safariでアプリを開く
  2. 共有ボタン（□に↑）→「ホーム画面に追加」
  3. ホーム画面のアイコンからアプリを起動
  4. 機内モードをON
  5. アプリが動くことを確認！
Android（Chrome）
  1. Chromeでアプリを開く
  2. 「インストール」バナーをタップ、または
  3. メニュー（⋮）→「アプリをインストール」
  4. ホーム画面のアイコンからアプリを起動
  5. 機内モードをON
  6. アプリが動くことを確認！

🔍 キャッシュの確認方法

Chrome/Edge開発者ツール
  1. F12キーで開発者ツールを開く
  2. 「Application」タブ（または「アプリケーション」）
  3. 左メニューの「Cache Storage」を展開
  4. chord-sheets-v2 というキャッシュを確認
  5. 中に全ファイルが保存されていることを確認

また、「Service Workers」セクションで登録状態も確認できます

⚠️ トラブルシューティング

問題: オフラインで動かない

解決方法:

  • HTTPSでアクセスしているか確認（GitHub Pages/Netlifyは自動的にHTTPS）
  • 初回アクセス時はオンラインである必要があります
  • Consoleエラーを確認（F12 → Console）
  • Service Workerが登録されているか確認（F12 → Application → Service Workers）
問題: 古いバージョンが表示される

解決方法:

  • ブラウザのキャッシュをクリア
  • Ctrl + Shift + R（スーパーリロード）
  • 開発者ツール → Application → Service Workers → Unregister

🎉 成功の確認

以下がすべて達成できればオフライン対応完了です！

  • ✅ 完全にネット接続を切った状態でアプリが開く
  • ✅ データの作成・編集・削除ができる
  • ✅ キー変更が動作する
  • ✅ スマホのホーム画面から普通のアプリのように起動できる
  • ✅ 圏外・地下鉄・飛行機の中でも使える

🎸 これで外出先でもいつでもどこでもコード譜を見られます！

📝 補足情報

  • データの保存先: LocalStorage（あなたのブラウザ内）
  • 容量制限: 通常5〜10MB（数百曲は余裕で保存可能）
  • 同期: なし（各デバイス独立）
  • プライバシー: データは外部に送信されません
