🔗 参照URL機能 使い方ガイド

U-FRETやChordWikiなどのコード譜サイトのURLを保存して、簡単に参照できる機能です。

✨ この機能でできること

✅ できること
  • U-FRET、ChordWiki などのURLを保存
  • ワンクリックで参照サイトを開く
  • 見ながら自分でコード譜を入力・カスタマイズ
  • 自分の好きなキーで保存
  • ページ区切りを自由に設定

📝 基本的な使い方

ステップ1: 参照サイトを見つける
  1. U-FRET (https://www.ufret.jp/) または ChordWiki (https://ja.chordwiki.org/) にアクセス
  2. 演奏したい曲を検索
  3. コード譜のページを開く
  4. ブラウザのアドレスバーからURLをコピー
例: U-FRETのURL
https://www.ufret.jp/song.php?data=12345

例: ChordWikiのURL
https://ja.chordwiki.org/wiki/曲名
ステップ2: アプリで新規楽曲を作成
  1. My Chord Sheetsアプリを開く
  2. 「新規作成」ボタンをクリック
  3. 曲名とアーティスト名を入力
  4. 「参照URL」欄に、コピーしたURLをペースト
  5. 「リンクを開く」をクリックして参照サイトを確認
ステップ3: コード譜を入力
  1. 参照サイトを見ながら、歌詞とコードを入力
  2. コードは [C] [Am] のように [ ] で囲む
  3. 自分の好きなキーに設定
  4. 必要に応じてページ区切り（---）を追加
  5. 保存
ステップ4: いつでも参照可能
  1. 楽曲を表示すると「参照元を開く」ボタンが表示されます
  2. クリックすると参照サイトが新しいタブで開きます
  3. 「あのコードどうだったっけ？」と思ったらすぐ確認できます

💡 便利な活用方法

🎸 複数サイトを比較

同じ曲でもサイトによってコード進行が違うことがあります。

  1. U-FRETとChordWikiの両方を確認
  2. 自分が弾きやすい方を選択
  3. または両方の良いところを組み合わせる
📚 リンク集として使う

将来演奏したい曲のURLをどんどん保存しておけます。

  • 曲名だけ入力してURLを保存
  • 歌詞は後で入力
  • 「やりたい曲リスト」として活用
🎹 カスタマイズして保存

参照サイトをベースに、自分流にアレンジ。

  • イントロやアウトロを追加
  • 間奏のコード進行を変更
  • 自分のアレンジメモを追加

⚠️ 重要：著作権について

なぜ自動コピーではなく手動入力？

著作権を尊重するためです。

  • 歌詞は著作物: 他サイトから自動的にコピーすることは著作権侵害になります
  • 手動入力はOK: 個人の練習用として、見ながら入力するのは問題ありません
  • 参照URLは記録用: 「このサイトを参考にした」という自分用の記録です
  • 共有には注意: 入力したコード譜を他人と共有する際は著作権に注意してください
📌 守るべきルール
  • ✅ 個人の練習用として使用
  • ✅ 参照サイトを見ながら手動で入力
  • ✅ 自分のアレンジや解釈を加える
  • ❌ 歌詞を丸ごとコピー&ペーストして公開
  • ❌ 参照サイトのコード譜をそのまま再配布

🎯 実践例

📖 ケース1: ライブで演奏する曲を準備
  1. U-FRETで「夜空ノムコウ」を検索
  2. URLをコピー: https://www.ufret.jp/song.php?data=12345
  3. アプリで新規作成、URLをペースト
  4. 参照サイトを見ながらコードを入力
  5. バンドで演奏するキー（D）に設定
  6. サビの前と2番の前でページ区切り
  7. 保存して、PDF出力
🎤 ケース2: カラオケの練習
  1. ChordWikiで好きな曲を検索
  2. URLを保存
  3. カラオケで歌いやすいキーに変更
  4. 自動スクロールで練習
  5. 「このコードどうだったっけ？」→「参照元を開く」で確認

❓ よくある質問

Q1: U-FRET以外のサイトも使える？

A: はい、どんなサイトのURLでも保存できます。楽器TAB譜サイト、YouTube動画のURLなども保存できます。

Q2: URLを保存しないといけない？

A: いいえ、オプションです。参照URLなしで、完全にオリジナルでコード譜を作成してもOKです。

Q3: 参照サイトが消えたらどうなる？

A: アプリ内に入力したコード譜は残ります。参照URLのリンクが切れるだけです。

Q4: 複数のURLを保存できる？

A: 1曲につき1つのURLです。ただし、歌詞の中にメモとして複数のURLを書くこともできます。

Q5: スマホでも使える？

A: はい！スマホのブラウザでU-FRETを開いて、URLをコピーして、アプリに貼り付けできます。

🌟 この機能のメリット

  • 🔍 すぐに参照できる: 元サイトを探し直す必要なし
  • 🎨 自由にカスタマイズ: キー変更、ページ区切り、アレンジ追加
  • 📱 オフラインでも使える: 一度入力すれば圏外でもOK
  • 📄 PDF出力可能: 印刷してライブや練習で使用
  • ⚖️ 著作権を尊重: 自動コピーではなく手動入力
  • 📚 一箇所に集約: よく使うコード譜サイトのリンク集に

🎸 便利に、そして著作権を尊重して使いましょう！
