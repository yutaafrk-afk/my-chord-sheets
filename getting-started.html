🗺️ 全体の流れ

1 ZIPファイルをダウンロード

このチャットからZIPファイルをダウンロードして解凍します。

2 無料ホスティングサービスにアップロード

🤔 デプロイ方法の選択

アプリを公開する方法は2つあります。どちらも完全無料です！

方法A: GitHub Pages おすすめ

メリット:

  • ✅ 完全無料、容量無制限
  • ✅ 広告なし
  • ✅ 自分でURL管理
  • ✅ プログラマー御用達

デメリット:

  • ⚠️ アカウント作成が必要
  • ⚠️ 少し手順が多い

こんな人におすすめ:

  • 長期的に使いたい
  • カスタムURLが欲しい

方法B: Netlify 最速

メリット:

  • ✅ ドラッグ&ドロップだけ
  • ✅ 3分で完了
  • ✅ 初心者に優しい

デメリット:

  • ⚠️ URLがランダム
  • ⚠️ 無料プランに制限あり

こんな人におすすめ:

  • とにかく今すぐ使いたい
  • 簡単な方が良い

🐙 方法A: GitHub Pages（詳細手順）

ステップ1: GitHubアカウントを作成

  1. https://github.com にアクセス
  2. 右上の「Sign up」をクリック
  3. メールアドレスを入力
  4. パスワードを作成（8文字以上、数字と記号を含む）
  5. ユーザー名を決める（例: yuta-music）
  6. メール認証を完了
💡 すでにアカウントがある場合は、ログインしてください

ステップ2: 新しいリポジトリを作成

  1. ログイン後、右上の「+」→「New repository」をクリック
  2. Repository name: my-chord-sheets と入力
    （任意の名前でOK。これがURLの一部になります）
  3. Public を選択（無料で使うため）
  4. 「Add a README file」はチェックしない
  5. 「Create repository」をクリック
📝 例: ユーザー名が「yuta-music」、リポジトリ名が「my-chord-sheets」の場合
→ 最終的なURLは「https://yuta-music.github.io/my-chord-sheets/」になります

ステップ3: ファイルをアップロード

  1. ダウンロードしたZIPファイル（chord-app-final-v5.zip）を解凍
  2. chord-app フォルダの中身すべてを選択
    （フォルダごとではなく、中のファイルすべて）
  3. GitHubのリポジトリページで「uploading an existing file」をクリック
  4. 選択したファイルをドラッグ&ドロップ
  5. 下にスクロールして「Commit changes」をクリック
⚠️ 重要: アップロードするファイル
以下のファイルがすべて含まれているか確認してください：
  • ✅ index.html
  • ✅ manifest.json
  • ✅ service-worker.js
  • ✅ icon-192.png
  • ✅ icon-512.png
  • ✅ その他のHTMLファイル（ガイドなど）

ステップ4: GitHub Pagesを有効化

  1. リポジトリページで「Settings」タブをクリック
  2. 左メニューの「Pages」をクリック
  3. Sourceで「Deploy from a branch」を選択
  4. Branchで「main」→「/(root)」を選択
  5. 「Save」をクリック
⏱️ 1〜2分待つと、ページの上部に
「Your site is live at https://ユーザー名.github.io/リポジトリ名/」
と表示されます

ステップ5: アクセスして確認

  1. 表示されたURLをクリック
  2. 「My Chord Sheets」の画面が表示されればOK！
  3. このURLをブックマークしておきましょう
🎉 完了！
これであなた専用のコード譜管理アプリが使えるようになりました！

⚡ 方法B: Netlify（詳細手順）

ステップ1: Netlifyアカウントを作成

  1. https://app.netlify.com/signup にアクセス
  2. 「Email」を選択（または GitHub/GitLab でサインアップ）
  3. メールアドレスとパスワードを入力
  4. メール認証を完了

ステップ2: サイトをデプロイ

  1. ログイン後、「Add new site」→「Deploy manually」をクリック
  2. ダウンロードしたZIPファイルを解凍
  3. chord-app フォルダごとをドラッグ&ドロップ
    （NetlifyはフォルダごとでOK）
  4. アップロード完了まで待つ（30秒〜1分）
🎊 自動的にランダムなURLが発行されます
例: https://wonderful-unicorn-123abc.netlify.app

ステップ3: URLを確認

  1. 発行されたURLをクリック
  2. アプリが表示されればOK！
  3. URLをブックマーク
💡 URL名を変更したい場合:
Site settings → Domain management → Options → Edit site name
で好きな名前に変更できます（例: yuta-chords.netlify.app）

🎸 アプリの使い始め方

初回アクセス時

1. オフライン対応の準備

初回アクセス時、自動的に全ファイルがキャッシュされます。これにより、2回目以降はオフラインでも使えるようになります。

2. 「インストール」バナーが表示される（場合がある）

「インストール」ボタンが表示されたら、クリックすると便利です（後からでもOK）

最初の楽曲を作成

  1. 右上の「新規作成」ボタンをクリック
  2. 曲名を入力（例: 夜空ノムコウ）
  3. アーティスト名を入力（例: SMAP）
  4. オリジナルキーを選択（例: C）
  5. 歌詞とコードを入力:
[C]あれから [Em]僕たちは [F]何かを信じて [G]きたかな [Am]夜空ノム[Em]コウには [F]明日が[G]もう待っ[C]ている
  1. 「保存」をクリック
🎉 最初のコード譜完成！
キー変更、自動スクロール、PDF出力などの機能を試してみましょう。

📱 スマホでアプリ化する方法

iPhone / iPad（Safari）

  1. SafariでアプリのURLを開く
  2. 下部の共有ボタン（□に↑）をタップ
  3. 「ホーム画面に追加」を選択
  4. 名前を確認して「追加」をタップ
  5. ホーム画面にアイコンが追加されます
🎵 音符のアイコンが表示されます
これをタップすると、普通のアプリのように起動します

Android（Chrome）

  1. ChromeでアプリのURLを開く
  2. 「インストール」バナーが表示されたらタップ
  3. または、右上のメニュー（⋮）→「アプリをインストール」
  4. 「インストール」をタップ
  5. ホーム画面とアプリドロワーにアイコンが追加されます

PC（Chrome / Edge）

  1. アプリのURLを開く
  2. アドレスバー右側の「インストール」アイコンをクリック
  3. 「インストール」をクリック
  4. デスクトップアプリとして起動できるようになります

🔧 トラブルシューティング

問題: ページが表示されない

原因と解決策:

  • GitHubの場合: GitHub Pagesが有効になっているか確認。Settings → Pages を確認
  • URLが間違っている: /chord-app/ が含まれていないか確認。正しいURLは https://ユーザー名.github.io/リポジトリ名/
  • ファイルが正しくアップロードされていない: リポジトリに index.html があるか確認

問題: 「404 Not Found」が表示される

解決策:

  • GitHub Pagesの設定で main ブランチと /(root) が選択されているか確認
  • 数分待ってから再度アクセス（反映に時間がかかる場合がある）
  • URLの最後に /index.html を付けてアクセスしてみる

問題: オフラインで動かない

解決策:

  • HTTPSでアクセスしているか確認（GitHub Pages/Netlifyは自動的にHTTPS）
  • 初回アクセス時はオンラインである必要があります
  • ブラウザのDevTools（F12）→ Application → Service Workers で登録されているか確認

問題: ホーム画面に追加できない

解決策:

  • iOSの場合: 必ずSafariを使用してください（Chromeでは追加できません）
  • Androidの場合: Chromeを使用してください
  • プライベートブラウジングモードでは追加できません

問題: データが消えた

原因と対策:

  • ブラウザのキャッシュをクリアするとデータが消えます
  • 定期的にPDF出力してバックアップを取ることをおすすめします
  • 重要な曲は紙に印刷しておくと安心です

以下すべてにチェックが入ればセットアップ完了です！

  • ☐ アプリのURLにアクセスできる

  • ☐ 楽曲を新規作成できる
