  └─ アプリが使えるようになります！

🔧 方法A: GitHub Pages（推奨）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ GitHubアカウント作成
   https://github.com でサインアップ

2️⃣ 新規リポジトリ作成
   Repository name: my-chord-sheets
   Public を選択

3️⃣ ファイルアップロード
   chord-app フォルダの中身すべてをアップロード
   （フォルダごとではなく、中のファイルすべて）

4️⃣ GitHub Pages有効化
   Settings → Pages
   Source: Deploy from a branch
   Branch: main / (root)
   Save をクリック

5️⃣ 完了！
   1〜2分後、URLが発行されます
   https://ユーザー名.github.io/リポジトリ名/


⚡ 方法B: Netlify（最速）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ Netlifyアカウント作成
   https://app.netlify.com/signup

2️⃣ デプロイ
   Add new site → Deploy manually
   chord-app フォルダをドラッグ&ドロップ

3️⃣ 完了！
   すぐにURLが発行されます
   https://ランダム文字列.netlify.app

⚠️ トラブルシューティング
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【ページが表示されない】
  → GitHub Pagesの設定を確認
  → 数分待ってから再度アクセス

【404 Not Found】
  → Settings → Pages の設定を確認
  → main ブランチと /(root) が選択されているか

【オフラインで動かない】
  → 初回はオンラインでアクセスが必要
  → HTTPSでアクセスしているか確認

【データが消えた】
  → ブラウザのキャッシュクリアで消えます
  → 定期的にPDF出力でバックアップを
