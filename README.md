# My Chord Sheets - コード譜管理アプリ

自分専用のコード譜管理Webアプリです。キー変更、編集、保存が簡単にできます。

## 🚀 デプロイ方法（完全無料）

### 方法1: GitHub Pages（おすすめ）

1. **GitHubアカウントを作成**（持っていない場合）
   - https://github.com にアクセス
   - 無料でアカウント作成

2. **新しいリポジトリを作成**
   - 右上の「+」→「New repository」
   - Repository name: `my-chord-sheets`（任意の名前でOK）
   - Public を選択
   - 「Create repository」をクリック

3. **ファイルをアップロード**
   - 「uploading an existing file」をクリック
   - このフォルダの全ファイルをドラッグ&ドロップ
     - index.html
     - manifest.json
     - service-worker.js
     - icon-192.png
     - icon-512.png
   - 「Commit changes」をクリック

4. **GitHub Pagesを有効化**
   - Settings（設定）→ Pages（左メニュー）
   - Source: 「Deploy from a branch」
   - Branch: 「main」→ 「/(root)」
   - 「Save」をクリック

5. **完成！**
   - 数分後、`https://あなたのユーザー名.github.io/my-chord-sheets/` でアクセス可能
   - このURLをブックマークしておけばいつでもアクセスできます

### 方法2: Netlify（もっと簡単）

1. **Netlifyアカウント作成**
   - https://netlify.com にアクセス
   - 無料でサインアップ

2. **ドラッグ&ドロップでデプロイ**
   - ログイン後、「Add new site」→「Deploy manually」
   - このフォルダをドラッグ&ドロップ
   - 自動的にURLが発行されます

3. **完成！**
   - `https://ランダムな文字列.netlify.app` のようなURLが発行されます
   - カスタムドメインも設定可能

## 📱 スマホアプリとして使う方法

### iOS（iPhone/iPad）

1. Safariでサイトを開く
2. 下部の「共有」ボタン（□に↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

→ ホーム画面にアプリアイコンが追加されます！

### Android

1. Chromeでサイトを開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

→ ホーム画面にアプリアイコンが追加されます！

### PC（Chrome/Edge）

1. ブラウザでサイトを開く
2. アドレスバー右側の「インストール」アイコンをクリック
3. 「インストール」をクリック

→ デスクトップアプリとして起動できます！

## 💡 使い方

### 📡 オフライン対応について

このアプリは**完全オフライン対応**です！

**初回アクセス時:**
1. インターネット接続が必要（ファイルをダウンロード）
2. 自動的に全てのファイルがキャッシュされます
3. 「アプリとしてインストール」バナーが表示されたらインストール推奨

**2回目以降:**
- インターネット不要で完全動作
- 飛行機の中でも、地下鉄でも、圏外でも使えます
- データはあなたのデバイスに保存されています

**オフライン状態の確認:**
- ヘッダー部分に「オンライン」「オフライン」の表示
- オフライン時は黄色いバナーで通知

### コードの入力方法
歌詞とコードを一緒に入力する際、コードは `[C]` のように角括弧で囲みます。

```
[C]君に会えた [Am]日から
[F]僕の世界は [G]変わった
[C]あの日の空の [Em]青さを
[F]今でも覚え[G]てる
```

### キー変更
- 表示画面の↑↓ボタンでキーを変更
- リセットボタンで元のキーに戻る
- ±11半音まで転調可能
- **NEW!** 「このキーを保存」ボタンで、よく使うキーを保存
- **NEW!** 保存したキーで次回自動的に開く

### 自動スクロール機能
- **NEW!** 演奏中に自動でスクロール
- 速度調整可能（0.5x〜5.0x）
- プリセットボタンで素早く速度変更
- 開始/停止ボタンで簡単操作

### PDF出力・印刷機能
- **NEW!** 横向きPDFで保存・印刷可能
- 現在のキー（移調後）で出力
- ページ区切り機能（`---`で改ページ）
- 見やすいレイアウトで印刷

### 参照URL機能
- **NEW!** U-FRET、ChordWikiなどのURLを保存
- ワンクリックで参照サイトを開く
- 見ながら自分でコード譜を入力
- 著作権を尊重しながら便利に使える

### データ保存
- 入力したコード譜はブラウザに自動保存
- サイトを閉じても次回アクセス時に残っています
- データはあなたのデバイスにのみ保存されます

## 🎯 新機能の使い方

### キーの記憶機能

各楽曲ごとに「よく使うキー」を保存できます。

**使い方:**
1. 楽曲を開く
2. ↑↓ボタンで好みのキーに変更
3. 「このキーを保存」ボタンをクリック
4. 次回この曲を開くと、保存したキーで表示されます

**便利な使い方:**
- 自分の音域に合わせたキーを保存
- バンドで演奏する際のキーを記録
- カラオケ用のキーを保存

### 自動スクロール機能

演奏中に自動でスクロールします。

**使い方:**
1. 楽曲を表示
2. 「開始」ボタンをクリック
3. スクロール速度を調整（↑↓ボタンまたはプリセット）
4. 演奏に合わせてスクロール
5. 「停止」ボタンで停止

**速度の目安:**
- 0.5x: ゆっくりしたバラード
- 1.0x: 標準的なテンポ
- 2.0x: アップテンポな曲
- 3.0x: 速い曲

**ヒント:**
- プリセットボタン（0.5x、1x、2x、3x）で素早く切り替え
- 演奏中でも速度変更可能
- スペースキーで停止/再開（予定）

### PDF出力・印刷機能

楽譜として印刷したり、タブレットで見るためのPDF保存ができます。

**使い方:**
1. 楽曲を表示
2. 好みのキーに変更（必要に応じて）
3. 「PDF出力」ボタンをクリック
4. PDFファイルがダウンロードされます

**ページ区切りの使い方:**
1. 歌詞編集画面で、改ページしたい場所に改行を入れる
2. `---` または `===` と入力
3. 保存すると、PDF出力時にそこで改ページされます

例:
```
[C]1番の歌詞が [Am]ここまで
[F]続きます [G]ねー

---

[C]2番の歌詞は [Am]新しいページから
[F]始まります [G]よー
```

**PDF出力の特徴:**
- 横向き（landscape）で見やすい
- 現在のキー（移調後）で出力
- ページ番号付き
- 各ページにタイトル・アーティスト・キー情報を表示
- 等幅フォントでコードが見やすい

**便利な使い方:**
- ライブ用の譜面を印刷
- タブレットにPDFを保存してオフライン閲覧
- バンドメンバーと共有
- セットリスト順にPDFを保存

### 参照URL機能

他サイトのコード譜を参照しながら、自分用にカスタマイズできます。

**使い方:**
1. U-FRETやChordWikiで曲を見つける
2. そのページのURLをコピー
3. 新規楽曲作成で「参照URL」欄にペースト
4. 「リンクを開く」で参照サイトを確認
5. 見ながら自分でコード譜を入力・カスタマイズ
6. 保存

**メリット:**
- 自分の好きなキーで保存
- 歌詞の構成を自由に編集
- ページ区切りを自分好みに設定
- オフラインでも使える
- 著作権を尊重（手動入力）

**対応サイト例:**
- U-FRET (https://www.ufret.jp/)
- ChordWiki (https://ja.chordwiki.org/)
- その他、任意のコード譜サイト

## 🆘 トラブルシューティング

### データが消えた
- ブラウザのキャッシュをクリアすると消えます
- 定期的にバックアップを取ることをおすすめします

### インストールボタンが表示されない
- HTTPSでアクセスしていることを確認
- GitHub PagesやNetlifyは自動的にHTTPSになります

### スマホでホーム画面に追加できない
- iOSはSafari、AndroidはChromeを使用してください
- プライベートモードでは追加できません
